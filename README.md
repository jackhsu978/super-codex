# Super Codex

A small browser platformer inspired by classic 8-bit side-scrollers. You play as the original Codex companion, running, jumping, collecting coins, stomping enemies, using power-ups, and clearing four short courses.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Controls

- Move: Arrow keys or A/D
- Jump: Up, W, or Space
- Run / spark shot: Shift
- Crouch / enter pipe: Down or S
- Spark shot alternate: X or K
- Start / pause: Enter or P
- Restart: R
